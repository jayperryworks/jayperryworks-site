# Jay Perry
[jayperryworks.com](http://jayperry.works)

A portfolio site and blog. Built with [Svelte/Sapper](https://svelte.dev) and [Node](https://nodejs.org).

## Run locally
1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`

## Deployment
Run `npm run export`
